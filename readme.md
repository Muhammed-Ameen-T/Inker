## 📖 Overview

`ameen-mvc` is a simple and efficient CLI tool designed to **quickly generate a full-stack MERN (MongoDB, Express.js, React, Node.js) project** with a structured **MVC architecture**. It helps developers set up a scalable, production-ready project within seconds.

---

## ✨ Features

✅ **MERN Stack Ready** - Generates a full-stack project with backend and frontend.  
✅ **MVC Architecture** - Organized folder structure for better maintainability.  
✅ **Windows-Compatible** - Ensures smooth setup on Windows systems.  
✅ **Pre-configured Scripts** - Includes development, production, and deployment scripts.  
✅ **Authentication Middleware** - JWT-based authentication setup included.  
✅ **MongoDB Configuration** - Pre-set for database connection with `.env` support.  
✅ **Git Initialization** - Initializes a Git repository for easy version control.
